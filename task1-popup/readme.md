##Псевдоэлемент Target и CSS3 transform
[demo](http://tocher.github.io/yandex-shri/task1-popup/v4.html)

Только CSS3
###Плюсы
- Размеры popup-а зависят от контента
- Только CSS
- Адаптивность
- Минимальный код

###Минусы
- Не работает в IE 8

- Может мешать работе других эффектов с transform
